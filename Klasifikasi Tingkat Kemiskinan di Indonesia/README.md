# Klasifikasi Kemiskinan di Indonesia
![Header](asset/header.jpg) <br>
Seperti yang kita tahu, Indonesia adalah negara kepulauan yang memiliki 284.889.720 jiwa penduduk (Februari 2025) dengan keberagaman kondisi sosial dan ekonomi di setiap wilayah. Saya memulai project ini dengan tujuan membuat analisis dan visualisasi pengaruh faktor ekonomi, sosial dan distribusi kemiskinan terhadap hasil akhir suatu daerah di Indonesia dikatakan miskin tidaknya.

### Langkah-langkah project
- Data collecting
- Data pre-processing
- Analisis & visualisasi
___
## Data Collecting
Pada proses data collecting (pengumpulan data) ini, saya menggunakan platform [Kaggle](https://www.kaggle.com/). Kaggle adalah website yang salah satu kegunaannya adalah untuk gudang dataset. Berikut ini adalah link dataset (raw data) yang saya gunakan pada project ini: [https://www.kaggle.com/datasets/ermila/klasifikasi-tingkat-kemiskinan-di-indonesia](https://www.kaggle.com/datasets/ermila/klasifikasi-tingkat-kemiskinan-di-indonesia).
